I bought an EcoFlow Smart Gateway and DPUX combo. The original intent was to use it for general backup and load shifting. I originally thought that Ecoflow's TOU would handle this but it really doesn't (it also doesn't support my utility's rate plan). So I decided to use AI to code my own app and this is the result. It took a lot of time to figure out the control logic for the devices (a lot of false paths and trial and error). While still a work in progress this has a functioning logic/commands and tracking functionality. Right now it is hard wired for ComEd's BESH rate.
